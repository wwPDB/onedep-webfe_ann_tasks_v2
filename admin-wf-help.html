Annotation Tasks Admin Options Help

Model File Upload

Provides upload option of model coordinate file. Format options RCSB-mmcif and RCSB-cifeps will be converted to PDBx-mmCIF after upload.

Experimental Data File Upload

Provides upload option of structure factor data file. The only supported format is PDBx/mmCIF. Files can be uploaded in Gzip format.

New Entry Option

Assign a deposition data set identifier and annoator for the entry to be added to the workflow system. Model and optionally structure factor data files must be uploaded before this option is visible. If the purge option is select the any existing archive data directory will be completely replaced by a new directory containing the uploaded model and structure factor data files.

Reset Workflow Entry Status Option

Reset the workflow status/progress for an entry. Enter the target deposition data set identifier for the entry to be reset. If the purge model files option is selected then all versions of the model/structure factor file beyond the first version will be removed. The workflow status is reset to the begining of the pipeline.

Reassign Workflow Entry Annotator Option

Reassign the annotator responsible for the entry. Enter the target deposition data set identifier and new annotator for the entry.
