Annotation Tasks Help Information

Model Upload

Provides upload option of model coordinate file. Various flavors of PDBx/mmCIF format are supported. Files with non-standard file names will be renamed by project conventions using the placeholder D_0000000000 data set identifier.

Dataset may also be identified by data set id code. The latest files in the archive corresponding to the identifier will be be downloaded.

Viewer 3D

View input model coordinate data and map using JSMol.

Assembly Task

This task manages the calculation, display and selection of oligomeric assemblies calculated using the Pisa application. Details of the selected assemblies are merged into the coordinate model file using the update option. Pisa Reference - E. Krissinel and K. Henrick (2007). Inference of macromolecular assemblies from crystalline state. J. Mol. Biol. 372, 774-797. 

	    Additional Pisa command line options include:

	    --lig-exclude='list' - exclude ligands in the given list when calculating the oligomeric state. The
            list should be a comma-separated, no spaces,case-sensitive set of ligand names. 

            The followin value as special meaning:

            (all)    : exclude all ligands

	    --lig=auto  - auto processing for ligands (default); ligands may be forcefully fixed on protein
            surface subject to task complexity and data available

	    --lig=fixed - fix ligands on protein surface

	    --lig=free  - do not fix ligands on protein surface.

Standard Annotation Tasks

Manage calculation of link, nucleic acid geometrical features, distant solvent and secondary structure.

Link Calculation

Calculation of links is performed using a module of the href="http://sw-tools.pdb.org/apps/MAXIT/index.html">MAXIT program. 
	    Link calculation command line options:   None at this time

Solvent Position Calculation

Calculation of solvent position is performed using the standard approach documented in the wwPDB data processing guidelines. Solvent is repositioned in the model coordinate data file and distant waters are tabulated in a validation category. 
	    Solvent calculation command line options:   None at this time

Secondary Structure

Secondary structure features are created using the algorithms originally implemented in the PROMOTIF program. PROMOTIF Reference: E. G. Hutchinson & J. M. Thornton "PROMOTIF - A program to id entify structural motifs in proteins" Protein Science 5:212-220 (1996). 
	    An optional topology file may be used to 

	    Example topology file:

	    SHEET 1
	    4 5 7
	    1 9 3 8 5 6
	    2 3 8 5 7
	    SHEET 5
	    4 5 7
	    1 9 3 8 5 6
	    2 3 8 5 7

Edit/View

View and edit updated model file containing calculated annotations.

Download

Download copies of the updated model file containing calculated annotations, coordinates of any calculated assemblies, analysis file created by the Pisa program, site environment file from getsite, and the log files from each of the annotation task calculations.
